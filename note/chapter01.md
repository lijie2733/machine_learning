## 人工智能三要素

+ 算法
+ 数据
+ 算力

CPU与GPU区别

## 人工智能、机器学习、深度学习关系

+ 机器学习是人工智能的一种实现方式
+ 深度学习是机器学习的一种方法发展而来

## 发展分支

+ 计算机视觉
+ 语音识别
+ 自然语言处理 文本挖掘分类
+ 机器翻译

## 机器学习流程

+ 数据输入
+ 数据处理
+ 特征工程
+ 机器学习
+ 模型评估（效果不好则回退至第二步）
+ 预测、分类

### 数据输入

#### *数据分类*

+ 离散值
+ 连续值

#### *数据分割*

+ 训练集
+ 测试集

#### *数据清洗*

+ 缺失值处理
+ 异常值处理

### 特征工程(Feature Enginerring)

使用专业背景知识和技巧处理数据，使得特征能在机器学习上发挥出更好的作用

#### *“数据和特征决定了机器学习的上限，而模型和算法只是逼近这个上线而已”*

#### *内容*

+ 特征提取
+ 特征预处理
+ 特征降维

### 算法分类

#### *监督学习*

+ 回归问题（预测连续值）
+ 分类问题（预测离散值）

#### *无监督学习*

输入数据无标记

#### *半监督学习*

训练集同时包含有标记数据和未标记数据
